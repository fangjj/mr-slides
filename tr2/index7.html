JSX Style Guide

Airbnb React/JSX Style Guide (English)

Airbnb React/JSX 编码规范（中文版）

  • 每个文件只写一个模块
  • 但是多个无状态模块可以放在单个文件中
  • 如果你的模块有内部状态或者是refs, 推荐使用 class extends React.Component
  • 没有状态或是没有引用refs， 推荐使用普通函数（非箭头函数）

DOM

Attach DOM Event Listener 

componentDidMount() {
  window.addEventListener('resize', this.handleResize);
}

componentWillUnmount() {
  window.removeEventListener('resize', this.handleResize);
}

DOM Updating


this.setState() //会调用 shouldComponentUpdate() 决定是否 re render

shouldComponentUpdate() //默认返回 true，re render，甚至 state 没有改变

1. Virtual DOM: re render，因为 React 不会自己去比较 state

2. Real DOM: React 负责在有更新时才 render 更新的部分

  render: function() {

  }
});
          

class MyComponent extends React.Component { // ES6
  render() {
    return <div className={this.props.className}/>;
  }
}
