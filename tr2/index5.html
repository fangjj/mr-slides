数据存储

  • 不使用任何数据库管理系统（DBMS），把数据保存在内存里或直接使用文件系统
  • CDN - 网络存储
  • SQL: MySQL, PostgreSQL，SQLite (Realm)
  • NoSQL: Redis，MongoDB，CouchDB, PouchDB


各类数据库比较

MongoDB

  • NoSQL: 数据库 (db)-> 集合(collection) -> 文档(document) -> field
  • SQL: 表 table -> row -> column

MongoDB

NoSQL Pros
  • 设计简单，方便迭代
  • 管理维护方便
  • BSON 和 JSON 数据匹配，比 SQL 更适合网络开发


NoSQL Cons
  • 功能简单
  • 没有事务

MongoDB 使用入门

MongoDB crud

  • Create - Collection.insert()
  • Read - Collection.find() / Collection.findOne()
  • Update - Collection.update()
  • Delete - Collection.remove()
